MY OWN PDF READER

Good day,
I wanted to have my own *.pdf content reader. I created it using the HTML command: embed.
Below are instructions and a demo for it



Commands:

  • page=10 – arbitrary page number to which the PDF viewer should scroll when opened
  • chapter=2 – scrolls to the given chapter
  • zoom=50 – zoom value from 1 to 100, indicates the percentage size of the document display in the browser
  • view=Fit - sets the display size so that the entire document fits in the browser, another value can be fitH, for display according to height
  • scrollbar=0 – disables the scrollbar, 1 enables it
  • toolbar=0 – disables the toolbar, 1 enables it
  • statusbar=0 – turns off the bottom bar, 1 turns it onnavpanes=0 – disables page previews in PDF, 1 enables it
  • navpanes=0 – disables page previews in PDF, 1 enables it
  • For the location of the file, you can put # in quotation marks.
  • If you use multiple commands use & between them.
